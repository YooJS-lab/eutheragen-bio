# Eutheragen Bio Website

Static bilingual landing page for GitHub Pages.

## Files
- `index.html` — page structure
- `styles.css` — layout, theme, responsive design
- `script.js` — English/Korean toggle and animated scientific background

## Deploy on GitHub Pages
1. Create a GitHub repository.
2. Upload all files in this folder to the repository root.
3. In GitHub, go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root` folder.
6. Save.
7. Your site will be published on the GitHub Pages URL.

## Notes
